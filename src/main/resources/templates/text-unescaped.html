text는 전달받은 값을 일반 문자열을 처리
utext(unescaped)는 전달받은 값을 태그가 있으면 태그로 적용
sample="< b >정답입니다.< /b >" 서버에서 전달했을 때
text "< b >정답입니다.< /b >"
utext < b >"정답입니다"< /b >

text와 utext 비교

  • th:text=
  • th:utext=

t h : inline에서 none은 해당 태그에 타임리프를 적용하지 않는다.

[[,,,]] 와 [(...)] 비교

  • [[...]][[${data}]]
  • [(...)][(${data})]
