오브젝트(DTO)를 이용해서 Map, List 형식으로 값을 전달
배열은 값만 연속적으로 저장
Map(List)은 키와 값으로 연속적으로 저장 (키 - 변수, 값)
여러 변수를 하나의 덩어리로 전달을 할 때 사용(레코드 단위)
오브젝트로 전달받은 값은 오브젝트명.변수명으로 사용
