if문을 이용해서 조건에 맞는 작업을 처리
th:if=${조건식} 조건을 만족하면 처리
th:unless=${조건식} else문과 비슷, 조건을 만족하지 않으면 처리
if문을 대체 가능한 명령이 th:switch문
th:switch=${변수}
th:case="비교값"
th:case="비교값"
th:case="비교값"
th:case="*" 해당하는 값없음 => default
